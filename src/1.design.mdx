# Design

## Colors

The component library provides a set of built-in default colors that can be used directly.

### SemanticColors

<Space.Grid columns={{ xs: 2, sm: 4, md: 4, lg: 4, xl: 5, xxl: 6 }} spacing={4}>
  {Object.entries(SemanticColors).map(([name, value]) => (
    <Card key={name} isOutline style={{ backgroundColor: value }}>
      <Space.Flex row style={{ alignItems: "center", justifyContent: "space-between" }}>
        <Typography.Text text={name} as="strong" style={{ fontSize: 10, color: "#ffffff" }} />
        <Typography.Code text={value} style={{ fontSize: 10 }} />
      </Space.Flex>
    </Card>
  ))}
</Space.Grid>

```tsx
// import { Card, SemanticColors } from "venomous-ui-react/utils";

<Card style={{ background: SemanticColors.success }} />
```
