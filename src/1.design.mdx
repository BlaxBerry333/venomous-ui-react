# Design

## Modes

The component library provides `light` and `dark` theme modes.

```tsx
import { Theme } from "venomous-ui-react/components";
import { THEME_MODES, type ThemeMode, getSystemThemeMode } from "venomous-ui-react/utils";

function Component() {
  const { themeMode, setThemeMode, toggleThemeMode, resetThemeMode, isDarkThemeMode } = Theme.useThemeMode();
  const isLightMode = themeMode === THEME_MODES.Light;
  const systemThemeMode: ThemeMode = getSystemThemeMode();

  return <>...</>;
}
```

## Breakpoints

The component library provides `xs`、`sm`、`md`、`lg`、`xl`、`xxl` breakpoints.

```tsx
import { Theme } from "venomous-ui-react/components";
import { THEME_BREAKPOINTS } from "venomous-ui-react/utils";

function Component() {
  const { screenSize, isLargerThanOrEqual } = Theme.useBreakpoints();
  const isXsScreen = screenSize === THEME_BREAKPOINTS.Xs;
  const isLargerThanXsScreen = isLargerThanOrEqual(THEME_BREAKPOINTS.Xs);

  return <>...</>;
}
```

## Colors

The component library provides a set of built-in default colors that can be used directly. There are two ways to access colors:

> Use the <Typography.Code text="Theme.useThemeMode()" /> hook to get the theme mode and then access the colors.

```tsx
import { Theme } from "venomous-ui-react/components";
import { TEXT_COLORS } from "venomous-ui-react/utils";

function Component() {
  const { themeMode } = Theme.useThemeMode();
  const primaryText = TEXT_COLORS[themeMode].primary;

  return <>...</>;
}
```

> Use the <Typography.Code text="useDesign()" /> hook to access the colors directly.

```tsx
import { useDesign } from "venomous-ui-react/hooks";

function Component() {
  const design = useDesign();
  const primaryText = design.TextColors.primary;

  return <>...</>;
}
```
