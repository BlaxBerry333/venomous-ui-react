# useThemeDesigns

用于获取当前 `ThemeMode` 对应的主题设计。若通过 `<Theme.Provider>` 设置了自定义主题设计则返回自定义主题设计，否则返回默认主题设计。

<Source
  language="tsx"
  dark
  code={`
"use client";

import { useThemeDesigns } from "venomous-ui-react/hooks";
`.trim()}
/>
