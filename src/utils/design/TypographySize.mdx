# TypographySize

The component library provides a group of built-in default typography sizes that can be used directly.

```tsx
import {
  TYPOGRAPHY_SIZES,
  TYPOGRAPHY_SIZE_NAME,
  type TypographySize,
  type TypographySizeName,
} from "venomous-ui-react/utils/design";
```
