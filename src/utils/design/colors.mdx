# Colors

The component library provides a series of built-in default colors that can be used directly.

```tsx
import {
  SEMANTIC_COLORS,
  type SemanticColorName,
  THEME_COLORS,
  type ThemeColorName,
  TEXT_COLORS,
  BACKGROUND_COLORS,
  BORDER_COLORS,
} from "venomous-ui-react/utils/design";
```
