# Venomous UI

<Typography.Code text="venomous-ui-react" /> is a React component library built entirely with React Hooks and JSX
inline-styles.

This library is a personal learning project aimed at exploring the architecture and abstraction of UI components in
React. It is not meant for production use, and thus, not suitable for commercial applications at this stage.

## Instal

```shell
# Install the library from Github repository:
% npm install git+https://github.com/BlaxBerry333/venomous-ui-react

# Additionally, install the required peer dependencies:
% npm install @iconify/react@5 react-hook-form@7 motion@12 sonner@2 zod@4
```

```json
{
  "peerDependencies": {
    "@iconify/react": "^5.2.1",

}
```

## How to import

```tsx
// 1st import style
import { Button, type ButtonProps, getColors, useThemeMode } from "venomous-ui-react";

// 2nd import style
import { Button, type ButtonProps } from "venomous-ui-react/components";
import { useThemeMode } from "venomous-ui-react/hooks";
import { getColors } from "venomous-ui-react/utils";

// 3rd import style
import { Buttons, type ButtonProps } from "venomous-ui-react/components/Button";
import useThemeMode from "venomous-ui-react/hooks/useThemeMode";
import getColors from "venomous-ui-react/utils/getColors";
```
