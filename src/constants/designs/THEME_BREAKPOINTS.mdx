# THEME_BREAKPOINTS

我准备了几个内置的断点及其对应的浏览器宽度范围，涵盖从超小屏到超大屏的各种设备。

<Space.Grid columns={{ XS: 1 }} spacing={16}>
  {Object.entries(THEME_BREAKPOINTS).map(([key, value]) => {
    const [min, max] = THEME_BREAKPOINT_RANGES[value];
    const rangeText = max === Infinity ? `${min}px ~` : `${min}px ~ ${max}px`;
    return (
      <Space.Flex key={key} spacing={16}>
        <Typography.Text as="code" text={`THEME_BREAKPOINTS.${key}`} />
        <Typography.Text as="code" text={rangeText} />
      </Space.Flex>
    );
  })}
</Space.Grid>
