# PALETTE_COLORS

我准备了几组内置的颜色，可以直接使用。

<Space.Flex column spacing={40}>
  {Object.entries(PALETTE_COLORS).map(([key, value]) => (
    <Space.Flex column key={key} style={{ flex: 1, backgroundColor: value }}>
      <Typography.Title as="h4" text={key} />
      <Space.Grid columns={{ XS: 1, SM: 2, MD: 3, LG: 3 }} spacing={16}>
        {Object.entries(value).map(([colorName, colorValue]) => (
          <Space.Flex
            column
            spacing={8}
            key={`${key}-${colorName}`}
            style={{ flex: 1, backgroundColor: colorValue, padding: "40px" }}
          >
            <Typography.Text as="code" text={colorValue} />
            <Typography.Text as="code" text={`PALETTE_COLORS.${key}[${colorName}]`} />
          </Space.Flex>
        ))}
      </Space.Grid>
    </Space.Flex>
  ))}
</Space.Flex>
