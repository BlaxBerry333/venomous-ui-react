# Progress.PageLoading

`<Progress.PageLoading>` is used to display the animated loading status when route page is rendering.

It is a secondary repackaging of `<Progress.LoadingBar>` and the loading bar will be displayed in the center of the screen.

```tsx
import React from "react";

const LazyComponent = React.lazy(() => import("...."));

function App() {
  return (
    <React.Suspense fallback={<Progress.PageLoading />}>
      <LazyComponent />
    </React.Suspense>
  );
}
```
