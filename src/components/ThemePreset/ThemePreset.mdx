# ThemeProvider

`<ThemeProvider>` is an optional utility component, but strongly recommended to use to achieve a consistent theme style.

- Initialize global `ThemeColor` and `ThemeMode` during App startup.
- Inject dynamic CSS styles into HTML ( such as scrollbars„ÄÅtext selection highlights )
