# Theme.InjectToHTML

`<Theme.InjectToHTML>` is an optional utility component, but strongly recommended to use to achieve a consistent theme style.

- Initialize global `ThemeColor` and `ThemeMode` during App startup.
- Inject dynamic CSS styles into HTML ( such as scrollbars„ÄÅtext selection highlights )

<Controls />

```tsx
import { Theme } from "venomous-ui-react/components";

function App() {
  return (
    <Theme.Provider>
      {/* App Content */}

      <Theme.InjectToHTML />
    </Theme.Provider>
  );
}
```
