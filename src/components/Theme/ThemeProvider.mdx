# Theme.Provider

`<Theme.Provider>` is used to wrap your App project to ensure correct rendering and theming behavior ( Because all components rely on a shared `React.Context` for styling tokens and theme awareness )

```tsx
import { Theme } from "venomous-ui-react/components";

function App() {
  return (
    <Theme.Provider>
      {/* App Content */}
      {/* Other Components */}
    </Theme.Provider>
  );
}
```
