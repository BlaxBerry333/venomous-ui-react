# getLighter

用于基于一个 Hex 颜色获取一个更亮的 Hex 颜色

```tsx
"use client";

import { getLighter } from "venomous-ui-react/tools/getLighter";

console.log(getLighter("#4CAF50", 0)); // #4CAF50
console.log(getLighter("#4CAF50", 0.5)); // #72FF78
console.log(getLighter("#4CAF50", 1)); // #FFFFFF
```
